API Reference

Complete API for AI-enhanced blog content, story management, and recording control. Serves FINAL digest data with AI enrichments, structured SEO, and discovery feeds (RSS, sitemap, blogs index).

Pipeline: PRE-CLEANED (raw) → FINAL (AI-enhanced). Blog endpoints return FINAL data.
GET/api/blog/{date}

Retrieve complete AI-enhanced blog data including story packets, GitHub events, and AI-generated content.

Path Parameters
date
string (YYYY-MM-DD)
Example
/api/blog/2025-08-27
GET/api/blog/{date}/markdown

Get raw markdown content for a specific date's blog post.

Path Parameters
date
string (YYYY-MM-DD)
GET/api/blog/{date}/digest

Get digest data for a specific date's blog post.

Path Parameters
date
string (YYYY-MM-DD)
GET/blogs

Get the blogs index listing all available blog posts. Served from Cloudflare Worker with enhanced caching.

GET/rss.xml

Get the RSS feed for all blog posts. Served from Cloudflare Worker with 24-hour edge caching.

GET/sitemap.xml

Get the XML sitemap for the entire site. Served from Cloudflare Worker with 24-hour edge caching.

GET/stories/{date}

List story packets for a given date.

Path Parameters
date
string (YYYY-MM-DD)
GET/api/assets/stories/{date}

List all story assets for a specific date.

Path Parameters
date
string (YYYY-MM-DD)
GET/api/assets/stories/{date}/{story_id}

Get all assets for a specific story.

Path Parameters
date
string (YYYY-MM-DD)
story_id
string (story identifier)
GET/api/assets/blog/{date}

Get all assets for a blog post.

Path Parameters
date
string (YYYY-MM-DD)
GET/assets/*

Serve static assets (images, videos, highlights) directly from R2 bucket via Cloudflare Worker.

Example Paths
Story thumbnails
/assets/stories/2025/08/27/story_20250827_pr34_01_intro.png
Video files
/assets/stories/2025/08/27/story_20250827_pr34.mp4
POST/control/record/start

Start recording for a story (requires token). Supports optional bounded mode.

Headers
Authorization
Bearer <CONTROL_API_TOKEN>
Request Body
story_id
string
date
string (YYYY-MM-DD, optional)
bounded
boolean (default: false)
POST/control/record/stop

Stop recording for a story (requires token).

Headers
Authorization
Bearer <CONTROL_API_TOKEN>
Request Body
story_id
string
date
string (YYYY-MM-DD, optional)
